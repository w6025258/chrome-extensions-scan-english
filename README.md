# Chrome 词汇助手扩展

这是一个基于 Manifest V3 标准的 Chrome 扩展程序，旨在辅助阅读和英语学习。

## 功能特性

1.  **页面英语生词分析**：
    *   点击浏览器工具栏上的扩展**图标**。
    *   扩展会自动分析当前网页的所有内容。
    *   **自动提取英语单词**并统计出现频率。
    *   **智能过滤**：自动隐藏 "the", "is", "and" 等常见停用词，只展示有学习价值的单词。
    *   **自动采集**：开启后，浏览网页时自动将新词加入生词本。

2.  **生词本与学习模式**：
    *   **生词管理**：将页面分析结果一键保存到生词本。
    *   **学习状态**：支持将单词标记为“已学会”（绿色）或“已忽略”（黑色）。
    *   **卡片记忆**：提供专注的单词卡片模式，点击卡片即刻显示 Google 翻译。
    *   **智能去重**：已学会或已忽略的单词，在新的页面分析中会自动隐藏。
    *   **容量限制**：生词本上限 1000 词，防止数据过载。

## 项目结构

*   `manifest.json` - 配置文件，定义了权限、Popup 和 Content Scripts。
*   `content.js` - 网页脚本，负责全页单词的提取分析。
*   `popup.html` - 点击扩展图标时显示的界面，展示分析结果。
*   `popup.js` - 负责处理 Popup 界面的交互逻辑。
*   `study.html` - 全屏学习页面，提供单词管理和卡片复习功能。
*   `study.js` - 学习页面的逻辑代码。

## 如何安装

1.  在你的电脑上创建一个新文件夹（例如 `vocabulary-helper`）。
2.  将所有代码文件保存到该文件夹中。
3.  （可选）添加图标文件 `icon16.png`, `icon48.png`, `icon128.png`。
4.  打开 Chrome 浏览器，访问 `chrome://extensions`。
5.  开启右上角的 **开发者模式 (Developer mode)**。
6.  点击 **加载已解压的扩展程序 (Load unpacked)** 并选择该文件夹。

## 常见问题

*   **点击图标显示“请刷新页面”**：如果是刚刚安装或更新了插件，需要刷新已打开的网页，content script 才能生效。
*   **无法在 Chrome 商店页面或新标签页使用**：这是 Chrome 的安全限制，插件只能在普通网页（http/https）上运行。
