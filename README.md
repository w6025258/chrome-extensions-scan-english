# Chrome 词汇助手扩展

这是一个基于 Manifest V3 标准的 Chrome 扩展程序，旨在辅助阅读和英语学习。

## 功能特性

1.  **页面英语生词分析**：
    *   点击浏览器工具栏上的扩展**图标**查看。
    *   **自动采集**：开启后，浏览网页时自动将新词加入生词本。
    *   **智能过滤**：自动隐藏 "the", "is", "and" 等常见停用词，以及代码变量、乱码等非自然语言单词。

2.  **生词本管理**：
    *   **状态管理**：将单词分类为“生词本”、“已学会”、“已忽略”。
    *   **智能去重**：已学会或已忽略的单词，在新的页面分析中会自动隐藏。
    *   **批量添加**：支持手动粘贴文章或单词列表，系统会自动提取并过滤。
    *   **数据统计**：实时显示各状态的单词数量。
    *   **容量限制**：生词本上限 1000 词，防止数据过载。

3.  **学习模式**：
    *   **列表模式**：浏览生词，支持按频率或时间排序。点击“翻译”图标查看释义，点击“喇叭”图标朗读。
    *   **卡片模式**：专注复习的 Flashcard 界面。
        *   点击卡片：显示 Google 翻译结果。
        *   语音朗读：支持美式英语发音。
        *   我会了 / 不会：掌握的单词自动移出复习队列。
    *   **Google 翻译集成**：内置气泡翻译和外部跳转链接。

## 项目结构

*   `manifest.json` - 配置文件，定义了权限、Popup 和 Content Scripts。
*   `content.js` - 网页脚本，负责全页单词的提取、过滤和自动保存。
*   `popup.html` - 点击扩展图标时显示的界面，展示当前页分析结果。
*   `popup.js` - Popup 交互逻辑。
*   `study.html` - 全屏学习页面，提供单词管理和复习功能。
*   `study.js` - 学习页面核心逻辑。

## 如何安装

1.  在你的电脑上创建一个新文件夹（例如 `vocabulary-helper`）。
2.  将 `manifest.json`, `content.js`, `popup.html`, `popup.js`, `study.html`, `study.js` 保存到该文件夹中。
    *   *(注意：请删除项目中可能存在的 `background.js` 和 `index.tsx` 文件，本版本无需使用)*
3.  （可选）添加图标文件 `icon16.png`, `icon48.png`, `icon128.png`。
4.  打开 Chrome 浏览器，访问 `chrome://extensions`。
5.  开启右上角的 **开发者模式 (Developer mode)**。
6.  点击 **加载已解压的扩展程序 (Load unpacked)** 并选择该文件夹。

## 常见问题

*   **翻译无法显示**：请检查网络连接，插件直接调用 Google Translate API，部分网络环境可能需要配置。
*   **点击图标显示“请刷新页面”**：安装或更新插件后，请刷新已打开的网页以注入新的脚本。
